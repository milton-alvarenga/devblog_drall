Estrutura básica de uma função pl/pgsql no PostgreSQL

Posted on November 25, 2013 by Alvarenga Milton — Comments

Um das formas de criar uma função no PostgreSQL utilizando pl/pgsql é utilizado a estrutura básica abaixo apresentada:

CREATE OR REPLACE FUNCTION <nome>() RETURNS VOID AS $$
   DECLARE
       <variaveis. Nao tendo, nao precisa desta area DECLARE>
   BEGIN;
       <CODIGO DA FUNCAO AQUI>
   END;
$$ LANGUAGE plpgsql

Esta função não recebe argumento e nem retorna nada.
